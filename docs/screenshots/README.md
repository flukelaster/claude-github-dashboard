# Screenshots

Drop PNG/JPG captures in this folder using the filenames below. The main [../../README.md](../../README.md) references them exactly.

Tip: capture at **1440×900** or larger on a **dark Edge/Chrome window** for consistency — the dashboard is designed monochrome-dark-friendly.

## Required

| File | Page | Route | Notes |
|---|---|---|---|
| `01-overview-dark.png` | Overview | `/` | Hero shot. Dark mode. Show KPI cards + daily cost chart. |
| `02-overview-light.png` | Overview | `/` | Same page, light mode. For dark/light comparison strip. |
| `03-usage.png` | Usage | `/usage` | Stacked daily-cost chart + token table. |
| `04-productivity.png` | Productivity | `/productivity` | LOC-per-$ trendline + scatter. |
| `05-repos.png` | Repos | `/repos` | Full table with +/−/net LOC + AI % column. |
| `06-repo-detail.png` | Repo detail | `/repos/:id` | Stat cards + commit timeline + PR list. |
| `07-languages.png` | Languages | `/languages` | Aggregate + per-repo cards with icons. |
| `08-sessions.png` | Sessions | `/sessions` | Recent sessions table. |
| `09-heatmap.png` | Heatmap | `/heatmap` | Day × hour grid. |
| `10-forecast.png` | Forecast | `/forecast` | Burn-rate cumulative projection. |
| `11-settings.png` | Settings | `/settings` | Sync state + GitHub token section. |
